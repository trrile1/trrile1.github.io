Route 66 in McLean County

Collection of historical photos, site information, and location data, presented using ArcGIS Story Maps (link in repo About information.)

Greenways of McLean County

ArcGIS Story Map exploration of parks, trails, and other green recreation areas in McLean County, Illinois.

HSTP Analysis

RStudio project which gathers American Community Survey data at various levels and processes into maps, charts, and tables. An RMarkdown file populates a report with the generated graphics and tables for easy access by other planners.

Amtrak Explorer

Leaflet.js web map application which displays minimum travel times between Amtrak Stations, derived from a GTFS file processed in R using tidytransit.

Rural Transportation

A visualization of an on-demand rural transportation network, constructed from schedule information using QGIS and incorporated into a Mapbox tileset.
